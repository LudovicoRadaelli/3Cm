Disequazioni risolvibili con il grafico dei segni

Esercizio A
Studiare il segno della seguente frazione al variare del valore di \(x\) \[ \dfrac{-6x^2 + 11x -3}{-4x + 2} \]

Soluzione:
Etichettiamo con il nome \(f(x)\) la frazione \(\dfrac{-6x^2 + 11x -3}{-4x + 2}\).

\(f(x) \gt 0\,\,\,\,\)

se \(\,\,\,\, \dfrac{1}{3} \lt x \lt \dfrac{1}{2}\,\,\,\) oppure \(\,\,\,x \gt \dfrac{3}{2}\)

\(f(x) \lt 0\,\,\,\,\)

se \(\,\,\,\, x \lt \dfrac{1}{3}\,\,\,\) oppure \(\dfrac{1}{2} \lt x \lt \dfrac{3}{2}\)

\(f(x) = 0\,\,\,\,\)

se \(\,\,\,\, x = \dfrac{1}{3}\,\,\,\) oppure \(\,\,\,x = \dfrac{3}{2}\)

\(f(x)\,\) non è definita

se \(\,\,\,\, x =\dfrac{1}{2}\)

Esercizio B
Studiare il segno del seguente prodotto di polinomi al variare del valore di \(x\) \[ (2x + 3)(-2x + 1)(x - 1) \]


Soluzione
Etichettiamo con il nome \(p(x)\) il prodotto \((2x + 3)(-2x + 1)(x - 1)\)

\(p(x) \gt 0\,\,\,\,\)

se \(\,\,\,\, x \lt -\dfrac{3}{2}\,\,\,\) oppure \(\,\,\,\dfrac{1}{2} \lt x \lt 1\)

\(p(x) \lt 0\,\,\,\,\)

se \(\,\,\,\, -\dfrac{3}{2} \lt x \lt \dfrac{1}{2} \,\,\,\) oppure \(\,\,\, x \gt 1\)

\(p(x) = 0\,\,\,\,\)

se \(\,\,\,\, x = -\dfrac{3}{2}\,\,\,\) oppure \(\,\,\,x = \dfrac{1}{2}\,\,\,\) oppure \(\,\,\,x = 1\)
