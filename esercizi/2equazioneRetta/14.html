Esercizio


Stabilire per quali valori di \(k\) le seguenti rette sono perpendicolari \[ \begin{align*} &r:\qquad5kx + (2k + 3)y - 1 = 0 \\\\ &s:\qquad6x - 5y - 4 = 0 \end{align*} \]
Soluzione
Le rette sono perpendicolari se \(k = \dfrac{3}{4}\)




Esercizio


Stabilire per quali valori di \(k\) le seguenti rette sono parallele \[ \begin{align*} &r:\qquad 4kx + (-7k + 1)y + 4 = 0 \\\\ &s:\qquad (2k - 1)x - 3ky + 2 = 0 \end{align*} \]
Soluzione
Le rette sono perpendicolari se \(k = \frac{9\pm \sqrt{73}}{4}\)




Esercizio


Stabilire per quali coordinate di \(k\) il punto \(P(k^2\,,\,\,5k + 2)\) appartiene alla retta \(r\) di equazione \(20x - 3y + 1 = 0\).

Soluzione
Le rette sono perpendicolari se \(\,\,k = -\frac{1}{4}\) oppure \(\,\,k = 1\).
